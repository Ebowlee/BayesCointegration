# 策略优化想法

记录暂时无法实施但值得未来尝试的优化想法。

## 1. 基于半衰期的动态冷却期

**当前问题**：固定10天的冷却期对所有配对一视同仁

**优化方案**：
- 根据配对的历史半衰期动态设置冷却期
- 公式：`pair_cooldown_days = 2 * half_life`
- 半衰期短的配对（如5天）冷却期为10天
- 半衰期长的配对（如7天）冷却期为14天

## 2. 基于半衰期的动态持仓期限

**当前问题**：固定30天的最大持仓期不够灵活

**优化方案**：
- 根据配对的半衰期设置最大持仓期
- 公式：`max_holding_days = 4-5 * half_life`
- 给予慢速回归的配对更多时间
- 避免快速配对占用资金过久
